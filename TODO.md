# Lode App - Funzionalità da Implementare in Futuro

## 1. Caricamento Multiplo "Intelligente" dall'App (Area Admin)
**Obiettivo:** Permettere di selezionare molteplici PDF contemporaneamente dal telefono, estrarre automaticamente nome e tonalità (similmente allo script Python su PC) e caricarli tutti in blocco usando un solo commit (tramite l'API GitHub Trees per evitare problemi di concorrenza).
**Priorità:** Alta (permetterà di dismettere completamente l'uso del PC).

## 5. Scalette "Volanti" Condivise
**Obiettivo:** Creare un sistema per generare scalette improvvisate durante le prove o gli incontri informali. Il leader seleziona i canti, l'app genera un codice univoco a 4 cifre (o un QR code) e gli altri musicisti lo inseriscono per ritrovarsi la scaletta sincronizzata sui propri tablet, senza dover modificare il Google Sheets ufficiale.
**Priorità:** Media (richiede un piccolo backend o database JSON esterno, es. Firebase).
